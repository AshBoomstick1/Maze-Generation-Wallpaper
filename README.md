# Installation
1. Download the ``org.kde.plasma.mazeGeneration`` folder.
2. Move the folder into the ``/home/[USER_NAME]/.local/share/plasma/wallpapers`` folder.
3. You should then see it as a wallpaper type in the wallpaper settings.
